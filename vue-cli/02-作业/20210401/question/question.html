  1. （20分） 利用脚手架工具生成一套自定义的vue初始化项目【项目名为首字母缩写加project例如 ： lt-project】

  2. （10分）在node环境中配置相关接口

    • 要求：
    • 配置一个登录接口
    • 接口地址："/login"
    • 请求类型：POST
    • 请求参数：
      account:string 登录账户
      password:string 登录密码
    • 回调参数：
      {
         state:200, //200表示账户密码正确 201：表示账户不存在 202：表示密码错误
         data:[], //返回对应账户可参看的学生数据，管理员返回所有学生的数据，学生则只放回自己的数据
         msg:'', //回调提示（登录成功；账户不存在；密码错误）
      }


  3. （20分）完成登录页面；

    • 请求方式：POST
    • 请求地址：根据自己配置的node.js同步
